# PromptDeck

Браузерный конструктор для создания презентаций с помощью AI. Два инструмента в одном репозитории.

---

## Файлы

### `prompt-builder.html`
Пошаговый визард для генерации промпта под любой AI-инструмент (Gamma, ChatGPT, Midjourney и др.).

**7 шагов:**
1. Тема презентации
2. Цветовая гамма — 25 кураторских + 15 случайных палитр
3. Плотность информации
4. Визуальный стиль
5. Компоновка слайдов
6. Шрифтовая пара — 20 вариантов
7. Количество слайдов

Перед выдачей промпта показывает **живой превью слайда** с выбранными цветами, шрифтом и компоновкой. Промпт генерируется на русском и английском.

---

### `presentation-builder.html`
Полноценный генератор презентаций прямо в браузере.

- Вводишь тему и/или свой текст
- Выбираешь цвет, стиль, шрифт, количество слайдов
- Claude API генерирует структуру и контент
- PptxGenJS собирает файл
- Скачивается готовый `.pptx`

Работает через встроенный Claude API (только в claude.ai).

---

## Стек

- Чистый HTML / CSS / JS — без фреймворков
- [PptxGenJS](https://gitbrent.github.io/PptxGenJS/) — генерация PPTX
- Claude API (`claude-sonnet-4-20250514`) — генерация контента

## Использование

Открой любой файл напрямую в браузере — установка не нужна.
